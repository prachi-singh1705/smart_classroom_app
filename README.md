# 🎓 Smart Classroom App

An AI-powered Smart Classroom Management System that simplifies classroom administration by combining attendance automation, assignment management, timetable scheduling, and role-based dashboards for teachers and students.

---

## 📌 Overview

Smart Classroom App is a web application designed to digitize classroom operations. It enables teachers to create virtual classrooms, manage sessions, assign work, and monitor student participation, while allowing students to join classrooms, access learning resources, view timetables, and submit assignments from a single platform.

The project focuses on improving classroom efficiency through automation and a user-friendly interface.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/prachi-singh1705/smart_classroom_app.git
```

### Move to Project Directory

```bash
cd smart_classroom_app
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```
